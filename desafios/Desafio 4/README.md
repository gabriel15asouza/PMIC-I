## Como usar (Quickstart):
### Requisitos:
- Python
- MySQL80

Criar banco de dados com MySQL como em "desafio4_db" e inserir senha em app.py da seguinte forma:

    app.config['MYSQL_PASSWORD'] = 'INSERIR_SENHA'
